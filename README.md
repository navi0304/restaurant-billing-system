# restaurant-billing-system
A simple GUI-based restaurant billing system developed using Python and Tkinter.  
The application allows users to select food items, enter quantities, and automatically generate a formatted bill receipt.

---

## Features

- Graphical User Interface using Tkinter
- Menu display with item prices
- Quantity input for each menu item
- Automatic bill calculation
- Receipt-style bill output
- Reset button to clear entries

---

## Technologies Used

- Python
- Tkinter (Python GUI library)
